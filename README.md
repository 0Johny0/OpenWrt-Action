# 描述

根据 [Use this template](https://github.com/P3TERX/Actions-OpenWrt/generate) 定制自用 `x86_64` 平台 OpenWrt 系统。

在原版系统下仅扩充以下插件达到最精简效果：

- OpenCxxxx
- snmpd
- ttyd
- quick start

同时修改内存及存储空间皆为512M。
